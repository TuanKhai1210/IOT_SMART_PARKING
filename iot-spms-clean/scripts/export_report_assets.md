# Export Report Assets

Use this folder layout for report assets:

```text
docs/assets/exported_diagrams/   rendered PlantUML images
docs/assets/ui_mockups/          UI mockup images
docs/assets/demo_screenshots/    D01 to D27 working demonstration screenshots
```

Recommended screenshot naming:

```text
D01.png
D02.png
...
D27.png
```
