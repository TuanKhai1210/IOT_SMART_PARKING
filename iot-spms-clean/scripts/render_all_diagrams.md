# Render All PlantUML Diagrams

Recommended output folder:

```text
docs/assets/exported_diagrams/
```

Example commands:

```bash
java -jar plantuml.jar diagrams/01_use_case_diagrams/*.puml
java -jar plantuml.jar diagrams/02_ui_flows/*.puml
java -jar plantuml.jar diagrams/03_sequence_diagrams/*.puml
java -jar plantuml.jar diagrams/04_activity_diagrams/*.puml
java -jar plantuml.jar diagrams/05_state_charts/*.puml
java -jar plantuml.jar diagrams/06_architecture_diagrams/*.puml
java -jar plantuml.jar diagrams/07_class_diagrams/*.puml
```

You may also render diagrams with a PlantUML extension in VS Code.
